# risk-rules-query

风险规则查询技能 - 从知识库中查询各类合同的风险规则

## 功能描述

从风险规则知识库中查询指定行业的风险规则，返回结构化的风险信息列表，包括：
- 风险项名称
- 严重级别（critical/warning/notice）
- 风险应对建议
- 相关法律依据

## 支持的行业类型

| 行业 | 知识库文件 |
|------|-----------|
| 采购 | 采购合同风险规则.xlsx |
| 租赁 | 租赁合同风险规则.xlsx |
| 服务 | 服务合同风险规则.xlsx |
| 通用 | 通用风险规则.xlsx |

## 使用方法

### 1. 在 Claude Code 中使用

当用户想要查看特定行业的风险规则时：
- "查看采购合同的风险规则"
- "租赁合同有哪些风险？"
- "给我看服务合同的风险规则"
- "列出所有风险规则"

### 2. 命令行调用

```bash
# 查询指定行业
python .claude/skills/risk-rules-query/src/queryEngine.py 采购

# 列出所有可用的风险规则类型
python .claude/skills/risk-rules-query/src/queryEngine.py --list
```
